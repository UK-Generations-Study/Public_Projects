Project title: Using routinely collected healthcare data in England to identify breast cancer recurrence

Description:
This R code was translated from the Stata code written by Lucy Kilburn and Victoria Hinder in The Institute of Cancer Research Clinical Trials and Statistics Unit (ICR-CTSU) for their paper titled: "Use of routinely collected health data (England) to identify subsequent disease-related events in patients with primary breast cancer: a practical alternative to hospital-based follow-up for breast cancer clinical trials" (DOI: 10.1186/s13063-025-09085-1).

The R code is a translation of the original Stata code, and was applied and tested to the Breast Cancer Now Generations Study cohort of women diagnosed with breast cancer.

ICR-CTSU contact: Lucy Kilburn (Lucy.Kilburn@icr.ac.uk)
